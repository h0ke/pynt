# Pynt

A Python client for [Open Beer Database](https://github.com/tristandunn/openbeerdatabase).

## Requirements

* Python 2.6+
* simplejson
* httplib2
* urllib

## Example

~~~ python
from pynt.pynt import Beer

# Get a beer by ID.
print Beer.get(1)
~~~

Checkout `example.py` for more.
